    Git Gud v0.1.0 GitGud.Maintainer View Source

    Repository maintainer schema and helper functions.

    A GitGud.Maintainer is primarly used to associate GitGud.User to GitGud.Repo.

    Each repository maintainer also has a permission defining which actions he is able to perform on the repository. Following permissions are available:

    • :read — can read and clone the repository.
    • :write — can read, clone and push to the repository.
    • :admin — can read, clone, push and administrate the repository.

    By default, a newly created repository maintainer has the :read permission. Use update_permission/2 to change a maintainer’s permission.
